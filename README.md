## ✨ About This Repository

**C-Codes** is a structured collection of C programming examples, practice problems, and topic-based code files.  
It is organized in a simple, chapter-wise style so that each concept can be studied, revised, and modified easily.

This repository is useful for:

- strengthening core C fundamentals
- practicing topic-by-topic programming
- revising exam-ready code patterns
- keeping small programs and experiments in one place

## 🧩 What You Will Find Here

### 📘 Chapter-wise C Practice
The `chapter 2` to `chapter 9` folders contain numbered C source files, arranged for topic-based study and revision.

### 🔁 Loops and Control Flow
The `ashikur sir/loops` folder is dedicated to loop-based practice and repeated logic patterns.

### ⚙️ Bitwise Operations
The `bitwise operators` folder is where low-level bit manipulation and operator-based experiments are kept.

### 🧪 Offline Assignments / Practice Sets
The `Offline_1-1(CSE-102)` folder contains multiple offline practice sets, separated into subfolders for better organization.

### 🧠 Class / Utility Examples
The `class` folder contains useful examples such as:
- matrix handling
- file handling
- pointer practice
- file copy logic

## 🚀 How to Use

1. Open the folder you want to study.
2. Read the code and understand the logic.
3. Compile the `.c` file using any C compiler.
4. Modify the code and experiment with new cases.
5. Practice repeatedly until the logic becomes natural.

Example:

```bash
gcc filename.c -o output
./output
```
